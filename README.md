# 🖱️ Explorando Drag & Drop

> Interatividade em outro nível — arrastar e soltar elementos com JavaScript puro.

---

## 🗂️ Sobre

O projeto **Explorando Drag & Drop** é um estudo prático sobre a API nativa de **Drag and Drop** do HTML5, explorando como tornar elementos da página arrastáveis, detectar zonas de soltar e criar interfaces interativas e intuitivas.

---

## 📖 Conteúdo abordado

- 🖱️ Atributo `draggable` no HTML
- 📡 Eventos de drag (`dragstart`, `dragend`, `dragover`, `dragleave`, `drop`)
- 📦 Transferência de dados com `dataTransfer`
- 🎯 Definição de zonas de destino (drop zones)
- ✨ Feedback visual durante o arrasto

---

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/Vinizapella/Explorando-Drag-Drop.git

# Acesse a pasta
cd Explorando-Drag-Drop
```

> Abra o arquivo `index.html` diretamente no navegador — sem necessidade de servidor! 🌐
